  • Admin actions

    Action: /api/v1.0/admin/

    Option Action: check - Check the spider status

    Option Action: turn_on - Turn on spiders

    Option Action: turn_off - Turn off spiders

    Option Field: spider

    e.g. https://localhost:5002/api/v1.0/admin/check

    e.g. https://localhost:5002/api/v1.0/admin/turn_on?spider=test_spider

    e.g. https://localhost:5002/api/v1.0/admin/turn_off?spider=test_spider
